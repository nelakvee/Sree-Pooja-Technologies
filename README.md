# Sree Pooja Technologies Website

Static GitHub Pages website for Sree Pooja Technologies, Shop No. 35, Ground Floor, Chenoy Trade Centre, Secunderabad.

## Files

- `index.html` - page structure and business content
- `styles.css` - responsive layout and visual design
- `script.js` - mobile navigation and WhatsApp quote form
- `assets/hero-laptop-printer-balanced-v4.png` - generated hero image inspired by the shop photos, balancing laptop repair, printer service and software diagnostics without people
- `assets/service-*.png` - generated visual cards for printer service, laptop repair, software troubleshooting, new sales and verified refurbished laptop inventory
- `assets/logo-sree-pooja-clean.png` - web-ready logo copy with transparent background

## GitHub Pages

Push these files to a GitHub repository, then enable GitHub Pages from the repository settings. Use the root folder as the publishing source.
